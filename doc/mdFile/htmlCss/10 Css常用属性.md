### 目前需要掌握的 CSS 属性

- font-size：文字大小
- color：前景色（文字颜色）
- background-color：背景色
- width ：宽度
- height：高度

### CSS 属性-background-color

- background-color 决定背景色

### CSS 属性-color

- color 属性用来设置文本内容的前景色
- 包括文字、装饰线、边框、外轮廓等的颜色
